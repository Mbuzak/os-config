# OS config
My OS configuration

## Goal
- OS memory less than 16 GB *(Arch Linux)*
- Nice looking environment *(KDE Plasma 6)*
- Efficient terminal *(Alacritty)*
- Minimal Vim config

## Installation
```
git clone git@github.com:Mbuzak/os-config.git ~/git/os-config
cd ~/git/os-config
./install.sh
```

- Config vim
Run `:PlugInstall` in vim
